## 📌 About the Project
This is a front-end clone of the TripAdvisor website built using HTML and CSS.  
The project replicates the design and layout of the original website, including travel sections, destinations, and user interface elements.

---

## 🛠️ Tech Stack
- HTML5  
- CSS3  

---

## ✨ Features
- Responsive homepage design  
- Travel destinations section  
- Clean UI similar to TripAdvisor  
- Image-based layout  
- Navigation bar and sections
